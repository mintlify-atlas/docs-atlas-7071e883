title: TypeAdapter

Type adapters provide a flexible way to perform validation and serialization based on a Python type.

A `TypeAdapter` instance exposes some of the functionality from `BaseModel` instance methods for types that do not have such methods (such as dataclasses, primitive types, and more).

## Class Definition

```python
class TypeAdapter(Generic[T])
```

<ParamField path="T" type="TypeVar">

## Usage Examples

### Basic Type Validation

```python
from pydantic import TypeAdapter

# Validate a list of integers
ta_list = TypeAdapter(list[int])
result = ta_list.validate_python(['1', '2', '3'])
print(result)
#> [1, 2, 3]

# Validate a dictionary
ta_dict = TypeAdapter(dict[str, int])
result = ta_dict.validate_python({'a': '1', 'b': '2'})
print(result)
#> {'a': 1, 'b': 2}
```

### JSON Validation

```python
from pydantic import TypeAdapter

ta = TypeAdapter(list[dict[str, int]])
json_data = '[{"a": 1}, {"b": 2}]'
result = ta.validate_json(json_data)
print(result)
#> [{'a': 1}, {'b': 2}]
```

### Custom Configuration

```python
from pydantic import TypeAdapter, ConfigDict

ta = TypeAdapter(
    list[str],
    config=ConfigDict(str_to_lower=True, str_strip_whitespace=True)
)

result = ta.validate_python(['  HELLO  ', '  WORLD  '])
print(result)
#> ['hello', 'world']
```

### Serialization

```python
from pydantic import TypeAdapter
from datetime import datetime

ta = TypeAdapter(list[datetime])
dates = [datetime(2024, 1, 1), datetime(2024, 1, 2)]

json_bytes = ta.dump_json(dates)
print(json_bytes)
#> b'["2024-01-01T00:00:00","2024-01-02T00:00:00"]'

python_obj = ta.dump_python(dates, mode='json')
print(python_obj)
#> ['2024-01-01T00:00:00', '2024-01-02T00:00:00']
```
